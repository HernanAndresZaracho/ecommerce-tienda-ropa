# E-Commerce - Tienda de Ropa

E-commerce modular desarrollado con TypeScript, ideal para adaptarse a distintos rubros.

## Tecnologías

### Backend

- **Node.js** + **Express** + **TypeScript**
- **MongoDB** + **Mongoose**
- **JWT** autenticación

### Frontend

- **React** + **Vite** + **TypeScript**
- **Axios** peticiones HTTP

## 🔧 Instalación y Uso

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
